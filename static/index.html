Available Feeds

Select one of the endpoints below:

  • /btc-usd.xml – Bitcoin price ATOM/XML feed
    Append a timezone parameter to the feed URL to get local times: ?tz=Europe/Madrid, ?tz=-6, etc. (numeric UTC offset or IANA timezone names). Example: /btc-usd.xml=+1
  • /aemet/mapa-isobaras.xml – AEMET isobars Atom/XML feed
